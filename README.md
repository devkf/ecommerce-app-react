# E-commerce-app react

## Installation de l'application

E-commerce app requires [Node.js](https://nodejs.org/) v16.17.0 to run.

Install the dependencies and devDependencies and start the app.

```sh
cd ecommerce-app-react
npm i
npm run start
```
